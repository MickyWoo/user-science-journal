Ebay: Usability Test

https://www.ebay.com/
    Basis of this site is an - Ecommerce Website.


Criteria Overview

  • Find out how three users ( males in their twenteis ) can navigate chosen site https://www.ebay.com/ and test them on time completion and relatively amount of clicks to buying and selling on the site.
    Test-script download page
  1. How easy it was to buy an item
    • Time Spent
    • ease of navigation ( amount of clicks average)
  2. How easy it was to Sell an item
    • Time Spent
    • ease of navigation ( amount of clicks)
  3. Collective overview on possiible improvments


User 1:

-- Buying experience --

  • Time Spent
      The user had a relative easy time buying an item, it just took 4 minutes to go from the search bar and find exactly what he/she wanted.
  • ease of navigation ( amount of clicks)
      The user took around around 20 clicks so, very fast and collective to get to checkout.

-- Selling experience --

  • Time Spent
      This was an experienced ebay user as I asked found out because the user spent just 5 mins and knew exactly where to go in less than 5 mins an item was for sale.
  • ease of navigation ( amount of clicks)
      The user was under 30 clicks prob a lot less if not for previous experience to make the item more likeable to be bought.

User 2

-- Buying experience --

  • Time Spent
      Was not a game of speed to the user, but did not really take that long to find an item and buy, which took around 5-6 minuits.
  • ease of navigation ( amount of clicks)
      This user was a bit more picky in searching for items because of the wide variety of results, so around 50 clicks before checking out.

-- Selling experience --

  • Time Spent
      The user spent around 15 mins to sell an item
  • ease of navigation ( amount of clicks)
      The user took about 50 clicks to complete putting an item for sale. Again, did not have any trouble, just took their time to fully assess the site and its tools given.

User 3

-- Buying experience --

  • Time Spent
      Took about 7 mins, was a completly new user to the site, but once the user figured the basics of the site and found the searchbar things went smooth
  • ease of navigation ( amount of clicks)
      This user was new to the site, around 56 clicks before soemthing was bought.

-- Selling experience --

  • Time Spent
      The user spent around 20 mins to sell an item because there was a bit of reading and certain process he has to follow to completly sell the item. One note I will say is that a new user is guided on how to properly sell an item on the site.
  • ease of navigation ( amount of clicks)
      Due to the new user guide, the amount of click was around 100+ but, it was straightforward the only issuse was the amount of experimenting the user did to completly sell the an item.

Overview

  • Each user who was familiar with the site knew the ins and outs to get from points A to point B relatively quickly, even someone who was new did take a bit longer, but was guided through and I talked to user 3 and he didn't have a bad experience. he told me things were presented in a good manner especially the guide so he would actually come back and use the site again. From my perspective each user was able to spend just a few minutes to buy an item and did not have that many obsticales navigating the site to the to their desired end point.

-- possible improvemnts --

  • Icons
      user 3 mentioned to me that a large icon for "buy" and "sell" centered above the search bar would be very direct instead of the sell letters in top-mid of the website
